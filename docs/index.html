How It Works

8 steps, fully automated

One command triggers the entire pipeline. No configuration, no surveys, no manual data entry. Just your M365 activity and AI analysis.

1
Harvest
Query your M365 activity
Runs 15 targeted WorkIQ queries across email, meetings, Teams chats, and documents for your chosen time range.
Email Meetings Teams Documents
2
Extract
Parse into structured signals
Each distinct repeating behavior becomes a signal with source, frequency, time cost, and participants. No data is invented — every number traces back to WorkIQ.
3
Cluster
Group signals into patterns
Signals that share source type, topic, action, or trigger-output chain are merged. A pattern needs 2+ signals — lone signals are dropped.
4
Verify
Confirm real attendance
Calendar invites don't equal attendance. Validates actual meeting participation through WorkIQ telemetry before building meeting-based patterns.
Attendance Check
5
Filter
Remove already-automated work
Checks every pattern against M365 built-in features (Teams Copilot Recap, Focused Inbox, Viva Insights). If it's already handled, it's not a skill candidate.
M365 Check Relevance Threshold
6
Score
Rate automation feasibility + value
12-item dual rubric: Automation Score (how rule-based?) and Value Score (how much does it cost you?). Composite score ranks every pattern.
Automation 0-100 Value 0-100
7
Dashboard
Generate interactive analytics
Self-contained HTML dashboard with ranked patterns, bubble chart, source breakdown, rubric details, and estimated time savings. Zero dependencies.
8
Build
Scaffold the skill with one command
Pick any candidate and run /skill-creator to generate a working Claude skill from the analysis.
